Object in Javascript

# In JS Object represents one of its data types. It is used to store various keyed collections and more complex entities. Objects can be created using the Object() constructor or the object literal syntax.

# According to W3schools, In JavaScript objects are king. If you understand objects, you understand JavaScript. This is literally true In JavaScript, almost "everything" is an object,except Primitives Data Types.

# Objects are reference data types.

# Arrays are good but not sufficient for real world data.

# Objects don't have index, They work in Key Value pairs like dictionary in Python,if you gone through it.

How to create objects

# We can simply create objects using object literals (curly braces {} )

Javascript 

const person = {
    name: "avinash",
    age: 20
}
How to access data from objects

1. by dot(.) notation method
2. by bracket [ ] methods

Javascript 

// By dot(.) method 

console.log(person.name); // avinash 
console.log(person.age); // 20

// by bracket [] method 

console.log(person["name"]); // avinash 
console.log(person["age"]); // 20
How to add new key value pairs (properties) to objects
Javascript 

// By dot(.) 

person.gender = "male";
console.log(person);
// {name: 'avinash', age: 20, gender: 'male'}

// by bracket [] 

person["hobbies"] = "music";
console.log(person);
// {name: 'avinash', age: 20, gender: 'male', hobbies: 'music'}
Difference b/w dot and bracket notation
Javascript 

const person = {
    name: "avinash",
    age: 20,
    hobbies: ["coding", "music", "sports"],
    "person salary": 10000
};

// Use dot(.) notation for single word 
console.log(person.hobbies);
// ['coding', 'music', 'sports']

// For keys more than single word we use bracket [] 
console.log(person["person salary"]);
// 10000
