Object Destructuring in Javascript

# What if i want to store a specific properties to a new variable, we have to do like below.

Javascript 

const band = {
    bandName: 'led zeppelin',
    famousSong: 'stairway to heaven',
    year: 1968,
    anotherFamousSong: 'kashmir'
};

const bandName = band.bandName;
console.log(bandName);   // led zeppelin

const famousSong = band.famousSong;
console.log(famousSong); // stairway to heaven

# But what if i am telling you that you can achieve this is in easy way. Here comes in Object Destructuring in use.

Javascript 

const {bandName,year} = band;

console.log(bandName); // led zeppelin 
console.log(year);     // 1968

# We can stored them in a new variable at the time of Destructuring it .

Javascript 

const {bandName: var1, year: var2} = band;

console.log(var1); // led zeppelin 
console.log(var2); // 1968

# Accessing rest of the object properties .

Javascript 

const {bandName,year,...restprops} = band;

console.log(restprops);
// {famousSong: 'stairway to heaven', anotherFamousSong: 'kashmir'}
Nested Destructuring
Javascript 

const users = [
    {userId: 1,firstName: 'Avinash',gender: 'male'}, 
    {userId: 2,firstName: 'Sumit',gender: 'male'}, 
    {userId: 3,firstName: 'Sonam',gender: 'female'}
];

const [user1, user2, user3] = users;

console.log(user1); 
// {userId: 1, firstName: 'Avinash', gender: 'male'}

console.log(user2); 
// {userId: 2, firstName: 'Sumit', gender: 'male'}

console.log(user3); 
// {userId: 3, firstName: 'Sonam', gender: 'female'}
Specific properties Destructuring
Javascript 

const [{firstName}, ,{gender}] = users;

console.log(firstName); // Avinash
console.log(gender);    // female
Change variable name
Javascript 

const [{firstName:user1firstName, userId}, , {gender:user3gender}] = users;

console.log(user1firstName); // Avinash 
console.log(userId);         // 1 
console.log(user3gender);    // female
