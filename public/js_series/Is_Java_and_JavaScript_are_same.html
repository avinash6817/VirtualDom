Is Java and JavaScript are same ?

# Even though these two languages have syntax like C , but they are completely a different languages.

Differences

● Java is a compiled programming languages. that means before the program run, the code needs to be translated into a machine code , so the computer can understand it.

● Js is an interpreted languages . in Browser , an interpreter will read the code and run it without needing to compile it first.

● java is used as a server side languages where as javascript is primary used as a client side (frontend) language. But javascript can also be used to create backend applications with Nodejs.

● Java is strongly typed language and variable must be declare first to use in program, JavaScript is weakly typed language and have more relaxed syntax and rules.

● Java program has file extension “.Java ” which is executed by JVM(Java Virtual Machine), JavaScript file has file extension “.js ” and it is interpreted but not compiled,every browser has the Javascript engine to execute the JS code.
