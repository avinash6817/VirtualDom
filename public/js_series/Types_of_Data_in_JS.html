Types of Data in Javascript.

# Types of DATA in JS

# Primitive Data Types

● String
● Number
● Boolean
● Null
● Undefined
● Symbols(ES6)

# Reference Data Types / Objects

● Arrays
● Object literals
● Functions
● Dates
● Anything Else...

# JS is a Dynamically Typed Language.

● The same variables can hold multiple value.
● we do not need to specify types.
● Types are associated with values not variables

# just use typeof to know about the data types.

Primitive Data Types
Javascript 

// String
const name = 'avinash kumar';
console.log(typeof name); // string

// String
const rollno = '12'; // this is also string
console.log(typeof rollno); // string


// Number
const age = 45;
console.log(typeof age); // number

// Boolean
const haskids = true;
console.log(typeof haskids); // boolean

// Null
const car = null;
console.log(typeof car); // object

// Undefined
let test;
console.log(typeof test); // undefined

// Symbol
const sym = Symbol();
console.log(typeof sym); // symbol
Reference Data Types
Javascript 

// Array
const hobbies = ['movies', 'music'];
console.log(typeof hobbies); // object


// Object literal
const address = {
    city: 'boston',
    state: 'london'
}
console.log(typeof address); // object

const today = new Date();
console.log(today); // you will get current time, date, according to your time zone.

console.log(typeof today); // object
