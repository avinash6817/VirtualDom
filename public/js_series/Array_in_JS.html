All you need to know about Array in Javascript

# Array is a data structure.

# It is ordered collection of multiple items, called Elements.

# Array in can contain elements with values of different data types.

Javascript 

const myArray = [6, "avinash", [50, 60, "hello"], true];
// we can do even nested array , as you can see we have string, number, boolean..

# Indexing always start from zero. this means that the first item in an Array is refered with zero index. second item has one index, last item has arraylength - 1.

# In js Array is object in typeof method.

We can create an Array by many ways in Javascript

● Using square brackets []
● Using Array constructor
● Using Array.of()
● Using Array.from()
● Using spread operator (...)

Javascript 

// Using square []
const items = ['phone', 'laptop', 'mouse', 'keyboard', 'charger', 'router', 'elcado'];

// Using Array constructor
const item1 = new Array('phone', 'laptop', 'mouse', 'keyboard', 'charger', 'router', 'elcado');

// Using Array.of().
const item2 = Array.of(10);
// Array of length 1 with Å¿o element in it -- [10]

// This will create an Array of length 10 empty items
const item3 = new Array(10);
// [ <10 empty items> ]

// This will create an Array of [2,4]
const item4 = new Array(2, 4);
Accessing Elements by its index
Javascript 

const items = ['phone', 'laptop', 'mouse', 'keyboard', 'charger', 'router', 'elcado'];

console.log(items[0]); // phone
console.log(items[3]); // keyboard
console.log(items[5]); // router
console.log(items[10]); // undefined;

// we can access backward indexes by length properties
console.log(items.length); // 7
console.log(items[items.length - 1]); // elcado
We can also loop over an Array using For or any loop
Javascript 

const myArray = [1, 2, 3, 4, 5];

for (let i = 0; i < myArray.length; i++) {
    console.log(myArray[i]);
}
// 1 2 3 4 5
Array Method

Array.isArray()
method determines whether the passed value is an Array or not, it return boolean.

# Syntax--Array.isArray(value)

Javascript 

const items = ['phone', 'laptop', 'mouse', 'keyboard', 'charger', 'router', 'elcado'];
const item1 = { age: 20 };
const item2 = 'Navbar';
const item3 = undefined;
const item4 = [];

console.log(Array.isArray(items));      // true
console.log(Array.isArray({ age: 20})); // false
console.log(Array.isArray('Navbar'));   // false
console.log(Array.isArray(undefined));  // false
console.log(Array.isArray([]));         // true

.concat()

Javascript 

const item1 = ['phone', 'laptop', 'mouse'];
const item2 = ['keyboard', 'charger', 'router'];
const item3 = ['headphone', 'bluetooth', 'wifi'];

const items = item1.concat(item2);
console.log(items);
// ['phone', 'laptop', 'mouse', 'keyboard', 'charger', 'router']

console.log(item1.concat(item2, item3));
// ['phone', 'laptop', 'mouse', 'keyboard', 'charger', 'router', 'headphone', 'bluetooth', 'wifi']

// NOTE -- array.concat(arri, arr2,....., arrN)

.join()
method joins all elements of the Array using a separator and returns a string. default separator is comma(,)

Javascript 

const items = ['phone', 'laptop', 'mouse', 'keyboard', 'charger', 'router', 'elcado'];

console.log(items.join('/'));
// phone/laptop/mouse/keyboard/charger/router/elcado

console.log(typeof items.join('7'));
// string

.fill()
method is used to change the the elements to a static value or a selected index.

# we can also pass argument to fill() method, first argument is the value we change with, second argument is the starting index, last argument is to determine where to stop filling.

# fill() changes the original Array

Javascript 

const items = ['phone', 'laptop', 'mouse', 'keyboard'];

console.log(items.fill('elcado'));
// ['elcado', 'elcado', 'elcado', 'elcado']

console.log(items.fill('elcado', 1, 4));
// ['phone', 'elcado', 'elcado', 'elcado']

.sort()
method sorts the elements of an array and returns sorted Array. The default sort() method converts the element types into strings and then sort them, see the below result.

Javascript 

const numbers = [55, 22, 88, 36, 100, 1, 66];
numbers.sort();

console.log(numbers); // (7)Â [1, 100, 22, 36, 55, 66, 88]

// default behavior converted into string, 
// so sorting is done on the basis of ASCII code

# The sort() method accepts an optional comparator function as an argument . you can write a comparator function and pass to the sort() method to override the default sorting behavior. In simple language sort() method can take callback function as input.

Javascript 

const fruits = ['Banana', 'Orange', 'Apple', 'Mango'];
fruits.sort();

console.log(fruits);
// ['Apple', 'Banana', 'Mango', 'Orange']

const numbers = [55, 22, 88, 36, 100, 1, 66]

function ascending(a, b) {
    return a - b;
}

console.log(numbers.sort(ascending)); 

// [1, 22, 36 55, 66, 88, 100]

.splice()
method can add, update and remove elements from an Array. splice() overwrites the original array.

# Syntax -- splice(index, number of items to be removed, New elements to be added)

Javascript 

const fruit1 = ["Banana", "Orange", "Apple", "Mango"];

// At index 2, add 2 elements 
fruit1.splice(2, 0, "Lemon", "Kiwi"); 
console.log(fruit1);  // ['Banana', 'Orange', 'Lemon', 'Kiwi', 'Apple', Mango']


const fruit2 = ["Banana", "Orange", "Apple", "Mango"];

// At index 2, add 2 elements, remove 1 
fruit2.splice(2, 1, "Lemon", "Kiwi"); 
console.log(fruit2);  // ['Banana', 'Orange', 'Lemon', 'Kiwi', 'Mango']


const fruit3 = ["Banana", "Orange", "Apple", "Mango", "Kiwi" ];

// At index 2, remove 2 items 
fruit3.splice(2, 2); 
console.log(fruit3);  // ['Banana', 'Orange', 'Kiwi']

.some()
also return boolean whether the value is found or not but it is based on atleast one Element in the array passing the condition is true/false (provided as a function)

# some() executes the function once for each element in the array_

# If it finds an array element where the function returns a true value, some() returns true (and does not check the remaining values) Otherwise it returns false.

# some() does not change the original array.

Javascript 

const age1 = [5, 11, 18, 20];

function drivingLicense(age) {
    return age >= 18;
}

console.log(age1.some(drivingLicense));  // true

const age2 = [5, 11, 15, 17];

function drivingLicense(age) {
    return age >= 18;
}

console.log(age2.some(drivingLicense));  // false

.find()
method returns the first matched element from the array that satisfies the condition in the function.

}

const found = myArray1.find(numberGreaterThan10);

// 12 -- first matched from Array

const myArray2 = [5, 12, 8, 130, 75, 44];

function numberGreaterThan50(elem) {
    return elem > 50;
}

const found = myArray2.find(numberGreaterThan50);
console.log( found); 
// 130 -- first matched from Array

# NOTE -- Dont expect 75

.at()
method used to access the elements using both positive and negative indexes using a single method.

Javascript 

const items = ['phone', 'laptop', 'mouse', 'router','elcado', ]

console.log(items.at(0));  // phone
console.log(items.at(1));  // laptop
console.log(items.at(2));  // mouse
console.log(items.at(3));  // router
console.log(items.at(4));  // elcado

console.log(items.at(-1));  // elcado
console.log(items.at(-2));  // router

.toString()
method returns a string with all array values separated by commas. toString() does not change the original array.

Javascript 

const item = ['phone', 'laptop', 'mouse', 'router', 'elcado' ];

const itemString = item.toString(); 
console.log(itemString); 
// phone, laptop, mouse, router, elcado
